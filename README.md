![License](https://img.shields.io/badge/license-MIT-blue.svg)
![React](https://img.shields.io/badge/React-19.0-blue)
![Vite](https://img.shields.io/badge/Vite-7.0-purple)
![Tailwind CSS](https://img.shields.io/badge/Tailwind_CSS-4.0-38bdf8)

A modern, responsive official website for **CSKS**, built to showcase events, gallery highlights, office bearers, and contact information. This project focuses on a premium user experience with dynamic animations and a clean, professional aesthetic.

## 🚀 Features

- **Responsive Design**: Fully optimized for desktop, tablet, and mobile devices.
- **Dynamic Event Showcase**: Interactive carousels and grids to display past and upcoming events.
- **Gallery**: Masonry-style image grid to showcase memorable moments.
- **Leadership Team**: Dedicated sections for Professional Execom, Student Leadership Team (SLT), and GSAC.
- **Contact Integration**: Easy-to-use contact form and location details.
- **Modern UI/UX**: Built with Tailwind CSS for rapid and beautiful styling.

## 🛠️ Tech Stack

- **Frontend Framework**: [React](https://react.dev/) (v19)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) & [PostCSS](https://postcss.org/)
- **Routing**: [React Router](https://reactrouter.com/)
- **Linting**: ESLint

## 📦 Installation

1.  **Clone the repository**
    ```bash
    git clone https://github.com/Vasif-abdulla/csks_website.git
    cd csks_website
    ```

2.  **Install dependencies**
    ```bash
    npm install
    # or
    yarn install
    ```

## 🏃‍♂️ Running the Project

To start the development server with Hot Module Replacement (HMR):

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) to view it in your browser.

## 🏗️ Building for Production

To create a production-ready build:

```bash
npm run build
```

To preview the production build locally:

```bash
npm run preview
```
